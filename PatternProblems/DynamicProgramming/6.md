- This problem is boolean dp
- We have two choices pick or not pick
- If we pick then we will check if the remaining sum is present in the remaining array or not
- If we do not pick then we will check if the sum is present in the remaining array or not
- If any of the above two choices return true then we will return true otherwise we will return false
- We can use memoization to store the results of the subproblems
- The time complexity of this approach is O(n*sum) where n is the number of elements in the array and sum is the target sum
- The space complexity of this approach is O(n*sum) for the memoization table and O(n) for the recursion stack in the worst case, so the overall space complexity is O(n*sum)
