$$S(i)=\sum_{j=0}^{n-1}\lvert nums[i]-nums[j]\rvert=(nums[i]\cdot i-\sum_{j=0}^{i-1}nums[j])+(\sum_{j=i+1}^{n-1}nums[j]-nums[i]\cdot(n-i-1))$$

The above formula can be derived from the fact that for all $j < i$, we have $nums[i] \geq nums[j]$ and for all $j > i$, we have $nums[i] \leq nums[j]$.

This is used to calculate the sum of absolute differences for each element in the array efficiently. By precomputing the prefix sums of the array, we can compute the required sums in constant time for each element, resulting in an overall time complexity of O(n).
