- Kruskal + DSU
- We can use Kruskal's algorithm to find the minimum spanning tree of the graph formed by the points. We will create a list of all edges (distances between points) and sort them by weight. Then, we will use a Disjoint Set Union (DSU) data structure to keep track of connected components and ensure that we do not create cycles while adding edges to the MST. The total weight of the edges in the MST will be the minimum cost to connect all points.
