- Multi source BFS
- Many oranges rotten, which can inturn rot other oranges around them at same time
- So we need do BFS in parallel for all the rotten oranges
- We can use a queue to store the rotten oranges and their time of rotting
- We can also keep track of the time taken to rot all the oranges
- We can also keep track of the number of fresh oranges left
- We can keep doing BFS until there are no more rotten oranges in the queue
