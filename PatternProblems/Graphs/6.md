- Why BFS fails?
BFS minimizes number of edges,
NOT total path weight

- What Dijkstra optimizes?
Shortest weighted distance
from source to all nodes

- Why priority queue?
To always expand the node with the smallest known distance next,
ensuring we find the shortest path efficiently.

- What PQ stores?
weight + node

- Time complexity?
O((V + E) log V)

- Dijkstra works ONLY when:
weights are non-negative
Negative weights break the greedy property.
